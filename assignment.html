Differentiate between ++i and i++ with Examples

Answers:

++i (Pre-increment):
When the pre-increment case (++i) is used in an expression, the original value is incremented by 1 before the expression is executed.
Example:
let i = 1;
++i;
let x = 10 * i;
console.log(x); // Output: 20
This happens because the original value of i has been incremented by 1 before the expression is executed.

i++ (Post-increment):
For the post-increment case (i++), when it is assigned to an expression, the original value is used in the expression, and then it is incremented by 1.
Example:
let i = 1;
i++;
let z = 10 * i;
console.log(z); // Output: 10
Here, the original value of i is used in the expression before it is incremented by 1.
In summary, i++ increments after execution, while ++i increments before execution.
